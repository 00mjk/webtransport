Steps to run this demo:
  1. Enable "enable-experimental-web-platform-features" in "about:flags" or sign up for the origin trial.
  2. Configure config.js and server with the settings your want.
  3. Run the server via "go run server.go".
  4. Load index.html, either via file://...index.html or by http://localhost:3030/
  5. When the page is open, open the JS console to check for errors.
